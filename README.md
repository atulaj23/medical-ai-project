# 🏥 Medical AI Coding Project

🚀 A Flask-based AI web application that predicts disease from symptoms and suggests ICD & CPT codes.

---

## 🔹 Features
- Symptom-based disease prediction
- ICD & CPT code suggestion
- Fuzzy matching for better accuracy
- Multi-symptom input support

---

## 🔹 Tech Stack
- Python
- Flask
- Pandas
- HTML, CSS
- FuzzyWuzzy
